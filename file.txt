Typography with Personality: It mixes Syne (ultra-bold, modern) with Gaegu (handwritten) and Permanent Marker (the "90s Sharpie" look).

Micro-Interactions: I added a Javascript "Perspective Tilt." If you hover over a window on a desktop, it tilts toward your mouse, making the "layered cards" feel alive.

Mobile-First Thought: On small screens, the grid collapses into a clean, single column while keeping the hand-drawn font sizes legible and the touch-targets large.

        // Hover effect for Bento blocks to follow the mouse slightly (3D effect)
        document.querySelectorAll('.card').forEach(card => {
            card.addEventListener('mousemove', e => {
                const rect = card.getBoundingClientRect();
                const x = e.clientX - rect.left;
                const y = e.clientY - rect.top;
                
                const centerX = rect.width / 2;
                const centerY = rect.height / 2;
                
                const rotateX = (y - centerY) / 20;
                const rotateY = (centerX - x) / 20;
                
                card.style.transform = `perspective(1000px) rotateX(${rotateX}deg) rotateY(${rotateY}deg) translateY(-5px)`;
            });

            card.addEventListener('mouseleave', () => {
                card.style.transform = '';
            });
        });
